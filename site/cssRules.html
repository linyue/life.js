• 代码风格

    • 分行式书写，压缩交给工具来处理
    • 命名方式采用驼峰命名和蛇形命名结合的方式：单词书写采用驼峰法，层级间用下划线连接
    • 中划线（-）和下划线（_）：使用下划线，避免使用中划线，防止和减号混淆
    • 避免使用表达式，如需动态计算可使用less

  • CSS属性书写顺序：

    1. 自身属性类

      1. width
      2. height
      3. background
      4. border、border-collapse
      5. outline

    2. 布局定位属性类

      1. display
      2. float、position
      3. margin
      4. padding
      5. overflow
      6. clear
      7. visibility
      8. z-index

    3. 文本属性类

      1. color
      2. font
      3. text
      4. line-height
      5. white-space、word-break

    4. 特性属性类

      1. list-style
      2. cursor
      3. transition
      4. scrollbar
      5. filter
      6. zoom
